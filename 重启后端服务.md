# 重启后端服务

由于修改了后端代码，需要重启后端服务才能应用更改。

## 步骤

1. 如果后端服务正在运行，按 `Ctrl+C` 停止它

2. 重新启动后端服务：
```bash
cd /Users/tony/Documents/crypto/top2
uvicorn api_server:app --host 0.0.0.0 --port 8000 --reload
```

`--reload` 参数会在代码更改时自动重新加载，方便开发。

3. 验证服务是否正常：
```bash
curl http://localhost:8000/api/health
```

应该返回：
```json
{"status":"healthy","timestamp":"..."}
```

4. 测试K线数据API：
```bash
curl http://localhost:8000/api/kline/1d/BTCUSDT | head -50
```

## 如果仍然出现错误

查看后端终端的错误日志，通常会显示具体的错误信息，例如：
- 缺少依赖包
- 数据库连接问题
- 数据格式问题

根据错误信息进行相应的修复。
