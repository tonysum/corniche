# 启动说明

## 前置要求

1. 确保后端服务已启动（`api_server.py`）
2. Node.js 和 npm 已安装

## 快速启动

### 1. 安装依赖（首次运行）

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端将在 http://localhost:3000 启动

### 3. 确保后端服务运行

在另一个终端窗口：

```bash
cd ..
uvicorn api_server:app --host 0.0.0.0 --port 8000
```

后端将在 http://localhost:8000 启动

## 使用流程

1. 打开浏览器访问 http://localhost:3000
2. 在"下载数据"标签页配置下载参数
3. 点击"开始下载"启动下载任务
4. 在"交易对列表"标签页查看已下载的交易对
5. 在"查看K线"标签页查询和查看K线数据

## 注意事项

- 下载任务是异步执行的，点击下载后会立即返回，实际下载在后台进行
- 如果后端服务地址不是 `http://localhost:8000`，请修改 `.env.local` 文件中的 `NEXT_PUBLIC_API_URL`
- 首次下载大量数据可能需要较长时间，请耐心等待
